# ESP32 MQTT Temperature Monitoring

This project demonstrates IoT based temperature monitoring using ESP32 and Raspberry Pi using MQTT protocol.

## Components
- ESP32
- Raspberry Pi
- WiFi
- Laptop

## Software
- Arduino IDE
- Mosquitto MQTT Broker
- Python 3
- Paho MQTT Library

## Working
ESP32 publishes simulated temperature data using MQTT. Raspberry Pi acts as broker and subscriber.

## Output
Temperature data is displayed on Raspberry Pi terminal.
